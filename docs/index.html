Features

Everything you need to run AI efficiently

AIRO handles the hard parts automatically so you can focus on what you're actually building.

🔍
Auto-detection
Detects your CPU, RAM, GPU (NVIDIA / AMD / Apple Silicon) silently on startup. Zero config required.
📊
Live dashboard
Real-time monitoring in your browser. See current use, best performance, and your target side by side.
⚡
Live tokens/sec
Measures actual tokens/sec from your running Ollama model. See exactly how fast your hardware is performing.
🎮
One-click optimize
AIRO calculates the best settings for your hardware. Press Y to apply CPU priority, GPU power mode, RAM swap, and Ollama context — all at once.
🌐
Three modes
Local machine, cloud API (OpenAI/Anthropic cost tracking), or remote cluster via SSH agent. Switch with Tab.
🔧
Works with anything
Ollama, llama.cpp, PyTorch, TensorFlow, or cloud APIs. AIRO is model-agnostic.
Get Started

Up and running in 60 seconds

Three commands. Then open your browser.

1
Clone and install
Works on Linux, Mac, and Windows. Requires Python 3.8+.
$ git clone https://github.com/crea-troy/airo && cd airo && bash install.sh
2
Start the web UI
AIRO auto-detects your hardware and starts a local server.
$ python airo.py web
3
Open in browser
See your live hardware dashboard. Press Y to apply AIRO's recommended settings.
http://localhost:7070

Business Model

How AIRO becomes a business

The open source version builds trust and users. The Pro version serves the customers who genuinely need more.

1. Open source grows users
Free local version gets shared on Reddit, HuggingFace, and GitHub. Researchers and developers adopt it. 1000 GitHub stars = credibility. This is how HashiCorp, Elastic, and GitLab started.
2. Research labs are the first customers
Physics labs, university AI groups, and HPC centers run AI on clusters with no dedicated ML engineer. They will pay for a simple dashboard that shows them how to use their hardware better.
3. Pro waitlist captures intent
This website has a "Join Pro waitlist" button. Every signup is a potential paying customer. 100 signups = enough to validate the Pro version is worth building.
4. Open core model
Local monitoring stays free forever. Multi-node cluster management, team analytics, and remote agents are Pro features. Companies with 10+ GPU nodes will pay €50-200/month easily.
